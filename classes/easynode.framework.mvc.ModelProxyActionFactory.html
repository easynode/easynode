获取代理模型del函数的Action原型类，返回的是一个动态的类，而不是一个实例。

Parameters:

  • conditions Object

    查询条件字段、比较表达式. Notation : { $fieldName1 : $expression $fieldName2 : $expression }

  • orderBy Array

    排序字段，参考：IConnection.list的orderBy参数，支持参数中使用__orderBy__参数修改默认排序

  • conditionPattern String

    条件组合模板，参考参考：IConnection.list的conditionPattern参数

  • view easynode.framework.mvc.View

    该Action使用的视图，默认为new JSONView()，如需要显示HTML页面，请传入 TemplateView实例。

  • l Object

    事件监听器，分别在开始创建前调用beforeList，在创建完成后调用afterList，这两个函数均是async函数， 并且会被bind到action实例上执行，因此可以在这两个函数中使用this引用到action。 在beforeList函数中，可以返回一个string表示一个新的conditionPattern，实际查询时将按新的conditionPattern 组织SQL逻辑。

                                                                     // example
                                                                 Notation & Example : {
                                                                         beforeList : function(model, actionCtx, conditions, pagination, orderBy, conditionPattern) {
                                                                                 return function * () {
                                                                                         return 'AND $pluginName$ OR $pluginVersion$';
                                                                                 };
                                                                         },

                                                                         afterList : function(model, actionCtx, result) {
                                                                                 return function * () {
                                                                                 };
                                                                         }
                                                                 }
