Parameters:

    字段名或字段实例， 当只传递name参数时，name被 认为是一个字段实例。 注意：字段命名请一律使用驼峰命名

  • type String

    字段类型, easynode.framework.mvc.ModelField.TYPE_*枚举

  • maxLength Int

    最大长度(in bytes)，默认0

  • defaultValue Object

    默认值，默认null

  • nullable Boolean

    可否为空,　默认true

  • comment String

    注释, 默认''（空字符串）

  • readonly Boolean

    是否为只读字段，只读字段只出现在查询和读取时，忽略其在写入时的行为
